# Community Threat Intelligence Platform

AI-powered security platform that detects scams, phishing, malicious links, and suspicious behavior in Discord communities.

## Quick Start

```powershell
# 1. Install dependencies
npm install

# 2. Generate Prisma client + push schema (SQLite, no Docker needed)
npm run db:push

# 3. Start backend + dashboard together
npm run dev

# 4. In another terminal, start the bot (needs Discord token)
npm run dev:bot
```

## All Commands

| Command | Description |
|---|---|
| `npm run setup` | Full first-time setup (env + install + db) |
| `npm run dev` | Start backend + dashboard together |
| `npm run dev:backend` | Start backend only |
| `npm run dev:bot` | Start Discord bot |
| `npm run dev:dashboard` | Start dashboard only |
| `npm run build` | Build all packages |
| `npm run db:push` | Push schema to database |
| `npm run db:generate` | Regenerate Prisma client |
| `npm run db:studio` | Open Prisma Studio (DB browser) |
| `npm run db:reset` | Reset database |
| `npm run deploy-commands` | Deploy Discord slash commands |

## Services

- **Backend API**: http://localhost:3001
- **Admin Dashboard**: http://localhost:3000
- **Discord Bot**: Runs as background process

## Environment Variables

Edit `packages/backend/.env`, `packages/bot/.env`, and `packages/dashboard/.env.local`:

| Variable | Where to Get |
|---|---|
| `OPENAI_API_KEY` | https://platform.openai.com/api-keys |
| `DISCORD_BOT_TOKEN` | Discord Developer Portal → Bot → Token |
| `DISCORD_CLIENT_ID` | Discord Developer Portal → General → Application ID |
| `DATABASE_URL` | Already set to SQLite (`file:./dev.db`) — works out of the box |
| `API_KEY` | Auto-generated by `npm run setup` |
| `JWT_SECRET` | Auto-generated by `npm run setup` |
| `ADMIN_USERNAME` / `ADMIN_PASSWORD` | Default: `admin` / `admin` |

## Discord Bot Setup

1. Create app at https://discord.com/developers/applications
2. Create bot, copy token → `DISCORD_BOT_TOKEN`
3. Enable **MESSAGE CONTENT INTENT** and **SERVER MEMBERS INTENT**
4. Invite bot to your server (OAuth2 → URL Generator → bot scope + Manage Messages permission)
5. Run `/setup` in your server

## Tech Stack

- Backend: Node.js + Express + TypeScript + Prisma + SQLite (dev) / PostgreSQL (prod)
- Bot: discord.js + TypeScript
- Dashboard: Next.js + Tailwind CSS
- AI: OpenAI API
